# 🧬 pkg/bio — Биомаркеры и Лабораторные Анализы

## 🎯 Философия

Биомаркеры — это **мост** между:
- **Энергетическим слоем** (чакры)
- **Физическим слоем** (органы, ткани)
- **Психологическим слоем** (эмоции, травмы)
│ Человек (Holistic) │
├─────────────────────────────────────────────────────────┤
│ Духовный │ Сахасрара ↔ Витамин D ↔ Смысл жизни │
├─────────────────────────────────────────────────────────┤
│ Психологический │ Эриксон ↔ Кортизол ↔ Травма │
├─────────────────────────────────────────────────────────┤
│ Энергетический │ Чакры ↔ Гормоны ↔ Энергия │
├─────────────────────────────────────────────────────────┤
│ Эндокринный │ Железы ↔ Гормоны ↔ Биохимия │
├─────────────────────────────────────────────────────────┤
│ Физический │ Органы ↔ Клетки ↔ Ткани │
└─────────────────────────────────────────────────────────┘
